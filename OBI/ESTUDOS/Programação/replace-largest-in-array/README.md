## Replace Largest in Array

Write a program that reads the elements of a 3x3 matrix, and replaces all occurrences of the largest value with -1 and prints the resulting matrix.

## Input example

    2
    7
    6
    9
    5
    1
    4
    3
    8

## Output example

    2 7 6
    -1 5 1
    4 3 8
